Kyzr

A full-stack Web-application which interfaced with an Android App that allowed users to trade virtual torches on their phones and lookup where their torch has been on the website. The goal of the project was to use as many buzzwords as possible. The server consisted of Python Flask, Apache, Debian, MongoDB.
